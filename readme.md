# Redmine Plus

Redmine Plus 是一个 Chrome 浏览器扩展，用于增强 Redmine 项目管理系统的使用体验。

## 功能介绍

### 项目快捷入口
在 Redmine 顶部导航栏添加自定义项目快捷链接，快速访问常用项目。

### 耗时跟踪快捷入口
在顶部导航栏添加本月/上月耗时和报表快捷入口，一键访问时间跟踪数据。

### 沉浸式 Markdown 编辑器
为问题描述编辑器提供全屏沉浸式编辑环境：
- 左右分栏布局，左侧编辑，右侧实时预览
- 支持附件图片预览
- 编辑器和预览区同步滚动

### 问题详情抽屉查看
在 Issue 列表的 ID 列旁添加"打开"按钮：
- 点击后在侧滑抽屉中查看详情
- 支持双击行快速打开
- 当前行高亮显示

### 时间报表图表
在时间报表页面生成可视化图表：
- 柱状图展示用户工时统计
- 支持按项目堆叠展示
- 自动识别表格结构，无需额外配置

## 安装方法

1. 打开 Chrome 浏览器，访问 `chrome://extensions/`
2. 开启右上角"开发者模式"
3. 点击"加载已解压的扩展程序"
4. 选择本项目目录

## 使用说明

安装后访问 Redmine 网站即可使用各项功能。点击浏览器工具栏的扩展图标打开设置中心，可配置各功能的启用状态。

## 技术栈

- Chrome Extension Manifest V3
- 原生 JavaScript (ES6+)
- HTML5 / CSS3
- Chart.js 4.5.0
